# 什么是仓库？
仓库（ Repository ）是集中存放镜像的地方。

## 公有仓库使用
- 你可以在 https://cloud.docker.com 免费注册一个 Docker 账号。
- docker login 登录
- docker logout 退出登录
- docker search 查找官方库中的镜像

docker search python
