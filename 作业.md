## DOCKERFILE 练习 (FlaskAppDemo)
### 制作webapp镜像文件并运行
- Docker build -t web-app:latest .   #根据当前文件夹下面的dockerfile创建镜像文件
- docker images                      #查看编译好的image
- Docker run -p 4000:80 web-app      #运行镜像文件并映射端口
- Docker ps                          #查看正在运行的容器
- 127.0.0.1:4000                     #从外部访问容器里面Web APP
